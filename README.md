# Z-Chat - Plataforma de Personagens AI

## Como fazer deploy no Vercel (GRÁTIS)

### Passo 1: Crie uma conta no GitHub
- Acesse: https://github.com
- Clique em "Sign up"

### Passo 2: Crie um novo repositório
- Clique em "New repository"
- Nome: z-chat
- Deixe PÚBLICO
- Clique em "Create repository"

### Passo 3: Suba os arquivos
- No repositório, clique em "uploading an existing file"
- Arraste TODOS os arquivos desta pasta
- Clique em "Commit changes"

### Passo 4: Conecte ao Vercel
- Acesse: https://vercel.com
- Clique em "Sign up with GitHub"
- Autorize o Vercel
- Clique em "Add New Project"
- Selecione o repositório "z-chat"
- Clique em "Deploy"

### Passo 5: Pronto!
- Seu site estará em: https://z-chat-seunome.vercel.app

## Funcionalidades
